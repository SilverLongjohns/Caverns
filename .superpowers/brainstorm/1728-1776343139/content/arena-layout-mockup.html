Layout Breakdown

  • Turn order bar (top) — shows initiative sequence, highlights active unit
  • Arena grid (center) — ASCII tiles with subtle grid borders, movement range highlighted in green
  • Unit panel (right) — party and enemy HP bars, class info, skull ratings
  • Action bar (bottom) — context-sensitive buttons, movement counter
  • Combat log (bottom strip) — last 3 combat messages, same as current
